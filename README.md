# BatPred Dev Home Assistant App
BatPred Dev Home Assistant App

This app can be used with Home Assistant to run the Scholdan BatPred dev fork without AppDaemon.

The code here can also be used to run Predbat outside Home Assistant (but still talking to Home Assistant)

See the documentation https://github.com/Scholdan/predbat_addon/blob/main/batpred-dev/DOCS.md
